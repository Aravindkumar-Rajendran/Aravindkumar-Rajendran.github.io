title: Dataset
---


Dataset is the collection of data used to train, validate and test the machine learning model. A dataset is divided into two parts.
- Training Set
- Test Set

Generally, the ratio between train and test sets would be 70:30 respectively. Sometimes, validation set will also be created for training set to validate the model during the training. Test set is used to evaluate the model after training and should not used for training.

Check this out - [Train and Test sets](https://developers.google.com/machine-learning/crash-course/training-and-test-sets/video-lecture)
